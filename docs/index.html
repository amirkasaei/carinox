Training-Free Best-of-N + Gradient Ascent Compositional Alignment

CARINOX

Reward-guided initial noise exploration (best-of-N) and refinement (gradient ascent) for compositional text-to-image generation.

Paper Code

Overview

Key ideas

  • Category-aware composite rewards (fixed set chosen via correlation with human scores).
  • Best-of-N exploration to avoid unlucky initializations and improve coverage.
  • Gradient ascent on initial noise with per-reward gradient clipping (Multi-Clip) + latent regularization.

Why it works

  • No single metric is best for all categories → combine complementary signals.
  • Discrete search (exploration) + continuous refinement (optimization) → robust & stable.
  • Clipping prevents any reward from dominating → preserves realism.

Best image by composite reward

Method

Pipeline

  1. Select a fixed reward combination per category from a correlation study.
  2. Sample N noises; generate; score; keep top-k seeds (best-of-N).
  3. For each kept seed, ascend in latent space with per-metric clipping and latent regularization.
  4. Return the image with the highest composite reward.
for t in 1..T:
  img = Gθ(ε, p)
  grads = [ clip(∂r_i/∂ε, τ) for r_i in rewards ]
  ε ← ε + η · ( Σ grads − λ_reg · ∂||ε||/∂ε )
